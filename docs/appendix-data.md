## 파일 위치
두 파일 모두 이 저장소 **최상위 폴더**에 있습니다.

```
modeudata-population-web/
├─ population_2026_05.csv   ← 인구 데이터
├─ dong_centroids.csv       ← 위경도(지도용)
├─ index.html               ← 웹앱
└─ docs/                    ← 이 교재
```

주피터/코랩 작업 폴더에서 바로 읽으려면 같은 폴더에 두거나 경로를 맞춰 주세요.

## 1. population_2026_05.csv

| 항목 | 내용 |
|---|---|
| 출처 | 행정안전부 주민등록 인구현황 (2026년 5월) |
| 행 | 3,918개 행정구역 (전국/시도/시군구/읍면동) |
| 열 | 204개 = `행정구역` + `코드` + 인구 202열 |

### 열 구조
```
행정구역          예: "서울특별시 강남구 역삼1동"
코드              10자리 행정동 코드 (지도 연결용)
남자_0세 ~ 남자_99세, 남자_100세이상      (101열)
여자_0세 ~ 여자_99세, 여자_100세이상      (101열)
```

### 빠른 로딩 코드
```python
import pandas as pd
df = pd.read_csv("population_2026_05.csv")
df.columns = [c.replace("﻿","").strip() for c in df.columns]
male_cols   = [f"남자_{a}세" for a in range(100)] + ["남자_100세이상"]
female_cols = [f"여자_{a}세" for a in range(100)] + ["여자_100세이상"]
```

> 원본을 행정안전부에서 직접 받으면 인코딩이 **CP949**, 숫자에 **천 단위 콤마**가 있을 수 있습니다.
> 그때는 `pd.read_csv(파일, encoding="cp949", thousands=",")` 로 읽으세요(3장 참고).

## 2. dong_centroids.csv

| 항목 | 내용 |
|---|---|
| 행 | 3,558개 행정동 |
| 열 | `코드`, `lat`(위도), `lon`(경도) |

### 빠른 로딩 코드
```python
cent = pd.read_csv("dong_centroids.csv")
cent.columns = [c.replace("﻿","").strip() for c in cent.columns]
cent["코드"] = cent["코드"].astype(str).str.strip()
```

`df["코드"]` 와 `cent["코드"]` 를 문자열로 맞춘 뒤 `merge` 하면 지도를 그릴 수 있습니다(10장).
