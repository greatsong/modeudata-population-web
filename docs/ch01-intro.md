## 1. 데이터 분석은 "질문에 답하는 일"

데이터 분석은 화려한 그래프를 그리는 일이 아니라, **궁금한 것에 데이터로 답하는 일**입니다.
우리의 질문은 이렇습니다.

> 🤔 *"전국 3,900여 개 동네를 인구 구조가 비슷한 것끼리 묶으면 어떤 무리가 나올까? 우리 동네는 어떤 무리에 속할까?"*

이 질문에 답하려면 데이터를 **모으고, 닦고, 들여다보고, 그리고, 학습시키는** 과정을 거쳐야 합니다.

## 2. 데이터 분석의 5단계 (이 교재의 지도)

데이터 과학에서 널리 쓰는 흐름(CRISP-DM)을 입문자용으로 단순화하면 이렇습니다.

```
 ① 수집        ② 정제          ③ 탐색·시각화      ④ 모델링(ML)       ⑤ 해석·공유
 데이터 모으기 → 깨끗하게 만들기 → 들여다보고 그리기 → 패턴 학습시키기 → 결과 읽고 나누기
   (2장)          (3장)            (4·5·6장)          (7·8·9장)         (10장)
```

이 다섯 단계는 한 번에 끝나지 않고 **빙글빙글 돕니다.** 그리다 보면 더 닦을 게 보이고,
모델을 돌리다 보면 새 질문이 생겨 다시 데이터를 모으죠. 그게 정상입니다.

## 3. 각 단계에서 우리가 할 일

| 단계 | 이 교재에서 하는 일 | 도구 |
|---|---|---|
| ① 수집 | 행정안전부 주민등록 인구 CSV 내려받기 | 웹브라우저 |
| ② 정제 | 인코딩·중복·이상한 행 처리, 표 만들기 | pandas |
| ③ 탐색·시각화 | 평균연령·고령화율 계산, 인구 피라미드 그리기 | pandas, matplotlib |
| ④ 모델링 | 동네를 특징 벡터로 바꾸고 **K-평균으로 군집화**, **PCA로 시각화** | scikit-learn |
| ⑤ 해석·공유 | 군집의 의미 읽기, 지도에 표시, 웹으로 배포 | matplotlib, (Plotly) |

## 4. "머신러닝"이라는 말에 겁먹지 마세요

이 교재의 머신러닝은 **비지도학습(unsupervised learning)** 중 **군집화(clustering)** 입니다.

- **지도학습**: "이건 개, 저건 고양이" 정답을 알려 주며 가르치는 것
- **비지도학습**: 정답 없이 "비슷한 것끼리 알아서 묶어 봐" 하는 것 ← **우리가 할 일**

우리는 "역삼동은 1번 무리"라고 정답을 준 적이 없습니다. 컴퓨터가 인구 구조의 **숫자 패턴만 보고**
스스로 비슷한 동네를 묶습니다. 그게 군집화이고, 그게 우리의 머신러닝입니다.
