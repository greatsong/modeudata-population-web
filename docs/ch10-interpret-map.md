## 1. 군집에 이름 붙이기

번호("군집 0")는 의미가 없습니다. 8장의 요약 표를 보고 **성격에 맞는 이름**을 지어 줍시다.

```python
summary = leaf.groupby("군집").agg(
    동네수=("행정구역","size"), 평균연령=("평균연령","mean"),
    평균총인구=("총인구","mean"), 평균소멸위험=("소멸위험지수","mean")
).round(1)
print(summary)
```

표를 보고 사람의 언어로 해석합니다(예시).

| 군집 | 특징 | 붙여 본 이름 |
|---|---|---|
| 평균연령 53 · 소멸위험 0.2 · 인구 적음 | 노인 많고 인구 적은 | 🌾 고령 시골 |
| 평균연령 38 · 인구 많음 · 0~9세 봉우리 | 젊은 부부+아이 | 🏗️ 신도시 육아 |
| 평균연령 41 · 20대 봉우리 | 청년 밀집 | 🎓 청년·대학가 |
| 평균연령 45 · 균형 | 평범한 도심 | 🏙️ 일반 도심 |

```python
names = {0:"고령 시골", 1:"신도시 육아", 2:"청년·대학가", 3:"일반 도심"}  # 표 보고 직접 채우기
leaf["군집이름"] = leaf["군집"].map(names)
```

> ⚠️ 이름은 **데이터를 보고** 붙이세요. 군집 번호는 실행마다 순서가 바뀔 수 있으니, 번호가 아니라 지표로 판단합니다.

## 2. 대표 동네 찾기 (군집의 얼굴)

각 군집 중심에 가장 가까운 동네 = 그 군집을 가장 잘 대표하는 곳.

```python
import numpy as np
for c in range(K):
    idx = np.where(labels == c)[0]
    center = X_sexage_s[idx].mean(0)
    nearest = idx[np.argmin(((X_sexage_s[idx] - center)**2).sum(1))]
    print(f"군집 {c+1} 대표:", leaf.loc[nearest, "행정구역"])
```

## 3. 지도에 군집 그리기

3장에서 불러온 위경도(`cent`)를 동네에 붙여 지도 위 점으로 찍습니다.

```python
import matplotlib.pyplot as plt
leaf["코드"] = leaf["코드"].astype(str).str.strip()
g = leaf.merge(cent, on="코드", how="inner")     # 위경도 결합

colors = ["#E8930C","#4C8DF6","#54A24B","#E45756","#B279A2","#3BA6A0","#F58518","#9C755F"]
fig, ax = plt.subplots(figsize=(7, 8))
for c in range(K):
    m = g["군집"] == c
    ax.scatter(g.loc[m, "lon"], g.loc[m, "lat"], s=8, alpha=0.6,
               color=colors[c % len(colors)], label=names.get(c, f"군집 {c+1}"))
ax.set_xlabel("경도"); ax.set_ylabel("위도"); ax.set_title("동네 군집 지도")
ax.legend(); ax.set_aspect(1.3); plt.tight_layout(); plt.show()
```

전국 단위로 돌리면 "고령 시골 군집은 지방에, 신도시 군집은 수도권 외곽에" 같은 **지리적 패턴**이 보입니다.
멀리 떨어진 동네가 같은 색이면 — 거리는 멀어도 인구 구조는 닮았다는 뜻이죠.

> 🗺️ 진짜 지도 배경(도로·해안선) 위에 그리고 싶다면 `folium` 라이브러리를 써 보세요.

## 4. 분석을 세상에 내보내기 (배포)

분석은 **공유될 때** 가치가 생깁니다. 세 가지 길이 있어요.

1. **노트북 공유**: 주피터 `.ipynb` 를 그대로 깃허브에 올리면 코드+결과가 함께 보입니다.
2. **정적 HTML**: `folium`·`plotly` 로 만든 그래프는 `.html` 로 저장해 누구에게나 보낼 수 있습니다.
3. **웹앱 + GitHub Pages**: 이 프로젝트처럼 `index.html` 하나로 만들어 GitHub Pages로 무료 배포.
   - 저장소 **Settings → Pages → Branch: main** 으로 켜면 `https://<아이디>.github.io/<저장소>/` 주소가 생깁니다.
   - 우리가 추가한 🧩 군집 분석 탭이 바로 이 교재 7~9장을 **사용자가 클릭으로 체험**하게 만든 결과예요.

## 5. 전체 여정 돌아보기

```
2장 수집 → 3장 정제 → 4장 지표 → 5장 시각화 → 6장 유사도
        → 7장 특징·표준화 → 8장 K-평균 → 9장 PCA → 10장 해석·지도·배포
```

여러분은 이제 **공공데이터 한 덩어리를 받아, 머신러닝으로 패턴을 찾고, 지도와 웹으로 공유하는** 전 과정을 해냈습니다.
같은 흐름은 인구뿐 아니라 상권·기후·교통 등 **어떤 데이터에도** 그대로 적용됩니다.
