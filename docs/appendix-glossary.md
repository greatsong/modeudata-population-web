# 부록 A. 용어 사전

← [목차로](README.md)

이 교재에 나온 용어를 한 줄로 정리했습니다. 막힐 때 찾아보세요.

## 데이터 다루기
| 용어 | 뜻 |
|---|---|
| **데이터프레임(DataFrame)** | pandas의 표. 엑셀 시트라고 생각하면 됩니다. |
| **인코딩(encoding)** | 글자를 컴퓨터가 저장하는 방식. UTF-8, CP949(EUC-KR) 등. 안 맞으면 한글이 깨짐. |
| **결측치(NaN)** | 비어 있는 값. `fillna`, `dropna` 로 처리. |
| **EDA** | 탐색적 데이터 분석. 모델 전에 데이터를 이리저리 들여다보는 단계. |

## 지표
| 용어 | 뜻 |
|---|---|
| **평균연령** | (나이×인구)의 합 ÷ 총인구. 가중평균. |
| **중위연령** | 인구를 한 줄로 세웠을 때 한가운데 사람의 나이. |
| **고령화율** | 65세 이상 인구 비율(%). |
| **소멸위험지수** | 20~39세 여성 ÷ 65세 이상 인구. 0.5 미만이면 소멸위험지역. |
| **노년부양비** | 65세 이상 ÷ 15~64세(생산연령) × 100. |

## 머신러닝
| 용어 | 뜻 |
|---|---|
| **특징(feature)** | 모델에 넣는 숫자. 동네를 대표하는 숫자 목록. |
| **특징 행렬 X** | 행=샘플(동네), 열=특징인 표. |
| **벡터(vector)** | 숫자의 목록. 공간의 한 점/방향으로 볼 수 있음. |
| **표준화(StandardScaler)** | 각 열을 평균 0·표준편차 1로 맞춤. 단위 통일. |
| **비지도학습** | 정답(라벨) 없이 패턴을 찾는 학습. 군집화가 대표. |
| **군집화(clustering)** | 비슷한 데이터끼리 무리로 묶기. |
| **K-평균(K-Means)** | 중심 K개를 잡고 "배정→중심이동"을 반복하는 군집 알고리즘. |
| **k-means++** | K-평균의 똑똑한 씨앗 초기화. 서로 멀리 떨어지게 시작. |
| **관성(inertia)** | 점과 자기 중심 사이 거리 제곱의 합. 작을수록 빽빽. |
| **엘보우(elbow)** | K를 늘리며 관성을 볼 때, 꺾이는 지점으로 적당한 K를 고르는 법. |
| **실루엣 점수** | 군집이 얼마나 또렷한지 -1~1로 나타낸 척도. |
| **차원 축소** | 많은 특징(고차원)을 적은 수(2~3)로 줄이기. |
| **PCA(주성분 분석)** | 정보(분산)가 가장 많은 방향을 새 축으로 삼아 차원 축소. |
| **주성분(PC1, PC2)** | PCA가 찾은 새 축. PC1이 정보가 가장 많음. |
| **코사인 유사도** | 두 벡터의 각도로 닮음을 잼. 1이면 같은 방향, 0이면 직각. |
