# 📚 바이브 코딩으로 배우는 데이터 분석 & 머신러닝 (HTML 입문 교재)

**코드 한 줄 안 쓰고, AI에게 시켜서** 전국 인구 데이터로
**수집 → 정제 → 분석 → 시각화 → 머신러닝(군집화) → 배포**까지 완주하는 입문 교재입니다.
pandas·numpy·설치 없이, 브라우저와 AI 대화창만 있으면 됩니다.

## 👉 교재 보기

- **라이브:** https://greatsong.github.io/modeudata-population-web/docs/
- **로컬:** 이 폴더의 [`index.html`](index.html) 을 더블클릭해 브라우저로 열기

## 특징
- 🎨 따뜻한 디자인의 단일 페이지 · 사이드바 목차 · 모바일 대응
- 📋 **복사 버튼**이 달린 프롬프트 박스 — 그대로 AI에 붙여넣어 실습
- 🧩 시각적 다이어그램(바이브 코딩 3박자, K-평균 4컷, PCA 비유, 인구 피라미드 등)
- ✅ 각 단계마다 `💬 이렇게 시켜요` · `👀 이렇게 확인해요` · `🔧 이렇게 고쳐요` · `🧪 직접 해보기`

## 구성 (0~10장 + 부록)

- 부록 A. 좋은 프롬프트 사전 · B. 꼭 알아야 할 개념 · C. 실습 데이터 안내
