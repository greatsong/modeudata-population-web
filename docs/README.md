# 📚 우리 동네 인구로 배우는 데이터 분석 & 머신러닝 (입문 교재)

> 『모두의 데이터 분석 with AI』 3장의 **웹 프로젝트**([../index.html](../index.html))를 교재로 풀어 쓴 입문서입니다.
> **데이터 수집 → 정제 → 탐색 → 시각화 → 머신러닝(군집화)** 까지, 실제 전국 인구 데이터로 한 흐름을 끝까지 따라갑니다.

이 교재는 **파이썬(pandas · matplotlib · scikit-learn)** 으로 실습합니다.
같은 로직이 실제 웹앱(`index.html`)에서는 자바스크립트로 어떻게 구현되었는지도 매 장 끝에서 짚어 줍니다.

## 이 교재가 답하는 질문

> "우리 동네는 **젊은 동네일까, 나이 든 동네일까?** 그리고 우리 동네와 **인구 구조가 닮은 동네**는 어디일까? 컴퓨터에게 전국 3,900여 개 동네를 **비슷한 것끼리 자동으로 묶어** 보라고 하면 어떤 그림이 나올까?"

이 한 질문을 풀기 위해 데이터 과학자가 거치는 전 과정을 그대로 밟습니다.
