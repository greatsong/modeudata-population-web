## 1. 무엇이 필요한가요

데이터 분석은 **연장(라이브러리)** 을 쓰는 일입니다. 우리가 쓸 연장은 네 개예요.

| 라이브러리 | 한마디로 | 비유 |
|---|---|---|
| **pandas** | 표(엑셀) 다루기 | 똑똑한 엑셀 |
| **numpy** | 숫자 계산 빠르게 | 계산기 엔진 |
| **matplotlib** | 그래프 그리기 | 도화지와 색연필 |
| **scikit-learn** | 머신러닝 | 학습하는 두뇌 |

## 4. 첫 코드: 데이터가 잘 열리는지 확인

```python
import pandas as pd

df = pd.read_csv("population_2026_05.csv")
print("행 수:", len(df))
print("열 수:", df.shape[1])
df.head(3)
```

다음과 같이 나오면 성공입니다.

```
행 수: 3918
열 수: 204
```

> 첫 열은 `행정구역`(예: 서울특별시 강남구 역삼1동), 둘째 열은 `코드`,
> 그다음부터 `남자_0세`, `여자_0세`, … `남자_100세이상`, `여자_100세이상` 까지
> **나이 한 살씩 남녀로 나뉜 202개 열**이 이어집니다.
